43 * <p>Implmentation of <b>Realm</b> that authenticates users via the <em>Java

61 * independence as possible from the underlying <code>LoginMethod</code>
62 * implementation executed by JAAS, the following policy is implemented by
63 * this Realm:</p>

66 *     <code>Subject</code> with at least one <code>Principal</code> instance
