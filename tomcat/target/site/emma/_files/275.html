65 * following replacement strings, for which the corresponding information
66 * from the specified Response is substituted:</p>
67 * <ul>
68 * <li><b>%a</b> - Remote IP address
69 * <li><b>%A</b> - Local IP address
70 * <li><b>%b</b> - Bytes sent, excluding HTTP headers, or '-' if no bytes
71 *     were sent
72 * <li><b>%B</b> - Bytes sent, excluding HTTP headers
73 * <li><b>%h</b> - Remote host name
74 * <li><b>%H</b> - Request protocol
75 * <li><b>%l</b> - Remote logical username from identd (always returns '-')
76 * <li><b>%m</b> - Request method
77 * <li><b>%p</b> - Local port
78 * <li><b>%q</b> - Query string (prepended with a '?' if it exists, otherwise
79 *     an empty string
80 * <li><b>%r</b> - First line of the request
81 * <li><b>%s</b> - HTTP status code of the response
82 * <li><b>%S</b> - User session ID
83 * <li><b>%t</b> - Date and time, in Common Log Format format
84 * <li><b>%u</b> - Remote user that was authenticated
85 * <li><b>%U</b> - Requested URL path
86 * <li><b>%v</b> - Local server name
87 * <li><b>%D</b> - Time taken to process the request, in millis
88 * <li><b>%T</b> - Time taken to process the request, in seconds
89 * <li><b>%I</b> - current Request thread name (can compare later with stacktraces)
90 * </ul>

92 * commonly utilized patterns:</p>

99 * <p>
100 * There is also support to write information from the cookie, incoming
101 * header, the Session or something else in the ServletRequest.<br>
102 * It is modeled after the
103 * <a href="http://httpd.apache.org/">Apache HTTP Server</a> log configuration
104 * syntax:</p>
105 * <ul>
106 * <li><code>%{xxx}i</code> for incoming headers
107 * <li><code>%{xxx}o</code> for outgoing response headers
108 * <li><code>%{xxx}c</code> for a specific cookie
109 * <li><code>%{xxx}r</code> xxx is an attribute in the ServletRequest
110 * <li><code>%{xxx}s</code> xxx is an attribute in the HttpSession
111 * </ul>
112 *
113 * <p>
114 * Log rotation can be on or off. This is dictated by the
115 * <code>rotatable</code> property.
116 * </p>
117 *
118 * <p>
119 * For UNIX users, another field called <code>checkExists</code> is also
120 * available. If set to true, the log file's existence will be checked before
121 * each logging. This way an external log rotator can move the file
122 * somewhere and Tomcat will start with a new file.
123 * </p>
124 *
125 * <p>
126 * For JMX junkies, a public method called <code>rotate</code> has
127 * been made available to allow you to tell this instance to move
128 * the existing log file to somewhere else and start writing a new log file.
129 * </p>
130 *
131 * <p>
132 * Conditional logging is also supported. This can be done with the
133 * <code>condition</code> property.
134 * If the value returned from ServletRequest.getAttribute(condition)
135 * yields a non-null value, the logging will be skipped.
136 * </p>
137 *
138 * <p>
139 * For extended attributes coming from a getAttribute() call,
140 * it is you responsibility to ensure there are no newline or
141 * control characters.

695    /**
696     * Log the specified message to the log file, switching files if the date
697     * has changed since the previous log call.
698     *
699     * @param message Message to be logged
700     */
701    public void log(String message) {
702        if (rotatable) {
703            // Only do a logfile switch check once a second, max.
704            long systime = System.currentTimeMillis();
705            if ((systime - rotationLastChecked) > 1000) {
706                synchronized(this) {
707                    if ((systime - rotationLastChecked) > 1000) {
708                        rotationLastChecked = systime;
709    
710                        String tsDate;
711                        // Check for a change of date
712                        tsDate = fileDateFormatter.format(new Date(systime));
713    
714                        // If the date has changed, switch log files
715                        if (!dateStamp.equals(tsDate)) {
716                            close();
717                            dateStamp = tsDate;
718                            open();
719                        }
720                    }
721                }
722            }
723        }
724        
725        /* In case something external rotated the file instead */
726        if (checkExists) {
727            synchronized (this) {
728                if (currentLogFile != null && !currentLogFile.exists()) {
729                    try {
730                        close();
731                    } catch (Throwable e) {
732                        log.info(sm.getString("accessLogValve.closeFail"), e);
733                    }
734 
735                    /* Make sure date is correct */
736                    dateStamp = fileDateFormatter.format(
737                            new Date(System.currentTimeMillis()));
738 
739                    open();
740                }
741            }
742        }
743 
744        // Log this message
745        synchronized(this) {
746            if (writer != null) {
747                writer.println(message);
748                if (!buffered) {
749                    writer.flush();
750                }
751            }
752        }
753 
754    }

757    /**
758     * Return the month abbreviation for the specified month, which must
759     * be a two-digit String.
760     *
761     * @param month Month number ("01" .. "12").
762     */
